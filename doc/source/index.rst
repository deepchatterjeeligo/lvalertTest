Welcome to lvalertTest's documentation!
=======================================

**lvalertTest** provides utilities for simulating gravitational-wave candidate events.
These can be uploaded to the production database (aka *GraceDb*) or local simulated databases (aka *FakeDb*).
In the latter case, facimiles of *lvalert* functionality are provided.
